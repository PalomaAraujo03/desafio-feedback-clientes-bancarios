# 📊 Desafio Criativo – Extraindo Insights do Feedback de Clientes Bancários

## 📖 Sobre o projeto

Este repositório contém a resolução do **Desafio Criativo** da **DIO (Digital Innovation One)**, cujo objetivo é desenvolver um **prompt estruturado** para orientar uma Inteligência Artificial na análise de feedbacks de clientes bancários.

Durante a atividade, o prompt foi construído de forma incremental, considerando intenção, contexto, critérios de análise e restrições, resultando em uma instrução clara e eficiente para gerar insights úteis para a tomada de decisão.

---

## 🎯 Objetivo

Criar um prompt capaz de orientar uma IA a:

* Analisar feedbacks de clientes bancários;
* Identificar padrões, reclamações e elogios;
* Classificar os comentários por tema e sentimento;
* Gerar insights e recomendações práticas;
* Apoiar a melhoria da experiência do cliente.

---

## 🛠️ Etapas desenvolvidas

### ✅ Passo 1 – Definição da intenção

Foi definido o objetivo da análise, o público que utilizará os resultados e o formato esperado da resposta.

### ✅ Passo 2 – Contexto e restrições

Foram especificados:

* Contexto da análise;
* Dados disponíveis;
* Critérios de classificação;
* Cuidados com dados sensíveis;
* Restrições para evitar conclusões sem evidências.

### ✅ Passo 3 – Prompt final

Todos os elementos foram reunidos em um único prompt, estruturado para orientar a IA na análise dos feedbacks de forma clara, objetiva e confiável.
